# Simple Web3 Token Sender

This is a beginner-friendly blockchain project.

This project helps you:
- Connect a crypto wallet
- Send native tokens to another address
- Check blockchain network
- See transaction hash

This project uses:
- HTML
- JavaScript
- ethers.js

## Features
- One-click wallet connect
- Send tokens with input fields
- Show network name
- Show transaction hash

## Requirements
- MetaMask installed
- Internet connection
- Some test tokens (use testnet)

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. Enter receiver address
5. Enter amount
6. Click "Send Token"
7. Confirm in MetaMask

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- config.js -> App settings
- provider.js -> Web3 provider
- connect.js -> Wallet connect
- network.js -> Network reader
- sender.js -> Send token logic
- ui.js -> Screen updates
- validate.js -> Input validation
- utils.js -> Helper functions

## Learning Goal
This project helps beginners learn:
- How blockchain transactions are sent
- How wallets sign transactions
- How Web3 apps work
